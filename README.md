# 健康旅行 App - 健康行程规划应用

## 功能特性

这是一个专为澳门设计的健康旅行行程规划应用，具有以下核心功能：

### 主要流程
1. **欢迎页面** - 包含"健康旅行"和"我在"字样，以及"开始"按钮
2. **用户信息收集**
   - 年龄输入
   - 性别选择（女性额外询问月经情况）
   - 慢性疾病询问（如果有，提供疾病大类选项）
3. **语言选择** - 支持繁体中文、简体中文、英语、韩语、日语、葡萄牙语
4. **澳门地图页面** - 交互式澳门地图
5. **路线生成** - 一键生成个性化旅行路线

### SDG理念融入
- **减少不平等** - 路线规划考虑无障碍设施，确保残障人士也能享受旅行
- **气候行动** - 优先推荐步行路线，并标注碳排放减少量

### 技术实现
- 纯HTML/CSS/JavaScript单文件应用
- 响应式设计
- 使用OpenWeatherMap API获取实时天气数据
- 集成澳门地图

## 使用说明

直接在浏览器中打开 `health-travel.html` 文件即可使用。
